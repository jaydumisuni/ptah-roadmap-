# WP12 Cross-Machine Invariants

**Status:** CANDIDATE

1. Observation is not a Finding.
2. Finding is a bounded reviewed interpretation, not universal truth.
3. Claim and Evidence remain separate.
4. Evidence Item binds exact Content, digest, collection Attempt and authority.
5. Evidence Bundle preserves item-level identity, coverage, privacy and limitations.
6. Scanner/rule/CVE/backend identifiers remain Aliases or evidence.
7. Severity, confidence, exploitability, impact and acceptance remain separate.
8. Validation Run uses new WP02 Attempts and exact environment evidence.
9. Review Decision cannot rewrite observations or evidence.
10. Accepted Risk expires and never erases the Finding.
11. Dispute preserves competing positions and evidence.
12. Disclosure requires explicit audience, authority and redaction policy.
13. Remediation Proposal is not a Patch.
14. Patch is not application or verified repair.
15. Remediation Run preserves backup, Attempt and uncertain outcome evidence.
16. Patch/application acknowledgement cannot satisfy Post-Fix Verification.
17. Reproduction Protocol freezes claim scope, environment and independence criteria.
18. Reproduction Request is not execution.
19. Reproduction Run uses new Activities/Operations/Attempts and environment generations.
20. Same cache, same mutable environment or same hidden authority cannot claim independence.
21. Reproduction Comparison is separate from Run outcome and original Claim.
22. Failed, negative, partial, contradicted and inconclusive results remain immutable.
23. Security tools remain workloads/adapters; they do not define Ptah Core identity.
24. Private evidence cannot leak through public reports, generated patches or proof bundles.
