## Recovered internal foundation

### Lumi Download Manager

- browser extension and browser-to-download handoff are product requirements;
- browser-originated downloads need background progress, pause/resume, checksums and destination handling;
- downloaded bytes should land through the Transfer/Object system rather than remain browser-temp files;
- optional engines and browser helpers should report capabilities honestly.

### Hunter runtime and bridges

- optional Facility/provider failures should degrade only dependent capabilities;
- specialist/browser work should remain behind a bridge instead of being absorbed into a monolithic controller;
- local file roots, secrets and external credentials need explicit protection;
- operation IDs, retries, attempts and proof receipts remain separate from tool output;
- online/local authority and safe source synchronization cannot be replaced by browser state.

### Ptah accepted architecture

- internet is normally available;
- restrictions come from caller/Workspace/deployment, not autonomous Ptah policy;
- browser is a Facility of the working world, not Hunter/Ptah reasoning;
- browser downloads, screenshots, traces and research outputs are Objects/Artifacts;
- direct human and compatible-system use must share one Browser contract;
- live lookup should be callable by external reasoning systems without embedding a reasoning council inside Ptah.

## Strong internal rules to preserve

1. Browser access is a general Workspace capability, not a reasoning identity.
2. Optional browser/research failure must not stop unrelated Activities.
3. Browser downloads use the existing Transfer/Object lifecycle.
4. Browser credentials/cookies remain secret references/protected data.
5. External systems decide why and what to research; Ptah executes and returns evidence.
6. Live research results remain claims linked to sources, not accepted truth automatically.
7. Browser actions follow the same operation/receipt/proof rules as every other side effect.
8. Public Ptah remains neutral and must not name private callers or internal councils.
